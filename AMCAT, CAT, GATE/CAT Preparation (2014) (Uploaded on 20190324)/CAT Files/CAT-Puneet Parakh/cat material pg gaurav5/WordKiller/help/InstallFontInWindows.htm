How to Install a Font in Windows

The font file will have the file extension .ttf or .fon.
The font file OSAFONT.FON in WordKiller font directory is for CBT (Computer Based Test) font.

1 Press the Start button

and click on Control Panel.
2 Once the Control Panel window is opened, select and double-click on Fonts.

This will call up the Fonts folder, which you can view in the formats Large Icons, List, Similarity or Details by clicking the appropriate button.
3

On the File select and click on Install New Font.... This will call up a dialog box..
4 Browse and select the folder containing the unzipped font file.

Select the font you wish to install. In the example shown this is . You should select OSAFONT.FON in WordKiller/font/ directory.

Then press OK.

Windows will then install the font for you and leave you with an updated list of the fonts installed on your computer.
