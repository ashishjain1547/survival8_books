
WordKiller is an Enhanced Rich Text Editor.

The core idea of WordKiller is to "kill" a word by changing its color from black to gray through double-clicking. :)


Example Files : 

1. example/BarronGREWordList.txt -- a sample word list text file
2. example/BarronGREWordList.rtf -- the transform result of BarronGREWordsList.txt by WordKiller
3. example/AWL.txt  -- Academic Word List

Font File:

4. font/OSAFONT.FON -- CBT (Computer Based Test) font file
